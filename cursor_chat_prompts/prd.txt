## README Instructions for Setup and Usage

### Environment Setup
1. Clone the repository `git clone <repo_url>`
2. Navigate to the project directory
3. Install Poetry: `pip install poetry`
4. Install dependencies: `poetry install`
5. Set up environment variables: `copy env.example .env`

### Running the Application
1. Start the FastAPI server: `poetry run uvicorn app.main:app --reload`
2. Access the API via `http://127.0.0.1:8000/`

### Running Tests
1. Run tests with `poetry run pytest` from the project root directory

### Configuration
1. Set up environment variables if needed (e.g., API keys, Supabase credentials)

**Note**: Supabase integration is planned for a future phase. All other requirements are met and tested. LLM credentials and World Bank API integration are fully functional.
